- 📑 Metodologia e Desenvolvimento 🛠️
- Este projeto foi desenvolvido com foco em boas práticas de programação e segurança de dados 🔐. Abaixo, detalho o uso de ferramentas externas conforme os critérios de avaliação:

- 🔍 Pesquisa Técnica e Auxílio para a Lógica (Stack Overflow) 📚
- A base lógica e a implementação de funcionalidades complexas foram desenvolvidas através de estudo e consulta técnica no Stack Overflow 💡, com foco em:

- 🎯 Lógica de Controle Estrita: Implementação avançada de scanf dentro de estruturas if para validação de dados em tempo real, garantindo a robustez da entrada de dados 📥.

- 📊 Manipulação de Estruturas: Estudo e aplicação prática de vetores para o armazenamento e processamento eficiente de informações visando um código mais rebuscado 🏗️.

- 🏗️ Arquitetura de Dados: Soluções para problemas de sintaxe e fluxos de decisão baseados em documentação da comunidade técnica 📑.

- 🤖 Uso de IA (Google Gemini) ✨
- A ferramenta de IA foi utilizada para formatação e organização estética 🎨:

- 📏 Ajuste de indentação e organização de chaves para facilitar a leitura.

- 🖥️ Padronização visual de menus para melhoria da interface (UI).

- 📖 Melhoria da Legibilidade do código visando o aperfeiçoamento no entendimento do sistema.
